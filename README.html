Search narwhal is a free open source browser engine for tkinter based browsers Brought to you by Zohan Haque and Search narwhal Github Org

Documentation

  1. Fork or clone the repo: Use git clone or download the ZIP and open it in Visual Studio.
  2. Don't install from pip: It is not recommended to install from pip due to potential errors. Always use the source.
  3. Download the SDK: Get it from this link.
  4. Study the example: Refer to example_browser.py for guidance. Mistakes are common without it.
  5. Build your browser: Use the structure of example_browser.py and add your own features.
  6. Compile with PyInstaller: Use pyinstaller --onefile your_browser.py.
  7. Move directories: After compiling, move the searchnarwhal/ folder into the dist/ folder to prevent crashes.
  8. Create an installer: Recommended tool is NSIS, though Inno Setup or others work too.
  9. Publish or keep private: Upload your browser to GitHub or keep it proprietary.
  10. You're done! Enjoy your own custom browser built with SearchNarwhal.
